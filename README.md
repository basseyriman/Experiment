# NEXUS — Experiment Lab by RimanTech

> **A premium interactive web experiment lab.** Built with cutting-edge CSS, Canvas 2D, and vanilla JavaScript. No frameworks. Just raw performance.

## ✨ Features

| Feature | Description |
|---|---|
| 🧠 **Neural Particle Engine** | Self-organizing particles simulating neural behavior |
| 🌊 **Fluid Dynamics Sim** | Real-time Navier-Stokes visualization |
| 🔮 **Quantum Visualizer** | Wave function probability clouds |
| 🌌 **Galaxy Generator** | Procedural 10K-star spiral galaxies |
| 🎵 **Audio Spectrum** | Web Audio FFT visualization |
| 🤖 **Generative Art AI** | Evolutionary color algorithms |
| 🎨 **Interactive Canvas** | 4 force modes, 4 color themes, real-time particle physics |
| ⌨️ **Live Terminal** | Auto-typing command simulation |
| 📊 **Animated Stats** | Counters, progress bars, system metrics |
| ✉️ **Contact Form** | Fully functional with toast notifications |
| 💫 **Custom Cursor** | Reactive cursor with glow and scale effects |
| ⚡ **Loading Screen** | Premium animated loader with progress bar |

---

## 🎨 Design Philosophy

- **Dark-first** design with deep cosmic color palette
- **Glassmorphism** cards with glow effects and backdrop blur
- **Micro-animations** on every interactive element
- **Particle networks** in the background (80 nodes, fully connected)
- **Responsive** — works on all screen sizes

---

## 🛠️ Tech Stack

```
HTML5          → Semantic structure, SEO-optimized
CSS3           → Variables, animations, glassmorphism
Vanilla JS     → Canvas 2D, Web Audio API, IntersectionObserver
Google Fonts   → Outfit + Space Mono
```
