## What Is This?

This is the official website for **RimanTech** — a premium tech agency that designs and builds web applications, AI-powered tools, and scalable digital products for ambitious businesses and founders.

The site is built to win clients. Every section has a purpose:

| Section | Purpose |
|---------|---------|
| **Hero** | Immediate clarity — who we are, what we do, why it matters |
| **Services** | 6 core services clearly defined with deliverables |
| **Work** | 4 case studies with real results (users, revenue, metrics) |
| **Process** | 4-step process that builds trust and removes risk |
| **Testimonials** | Social proof from real client types |
| **Pricing** | Transparent investment tiers (£3,500 — Enterprise) |
| **Contact** | Fully functional enquiry form |
